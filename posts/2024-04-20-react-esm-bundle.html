--- { "title": "React ESM bundle", "description": "Creating ESM Bundles of React and ReactDOM for Browser Usage with Import Maps.", "layout": "page", "collection": "notes", "permalink": "/notes/react-esm-bundle.html" } --- {{>>layout.html}} {{>>markup-paragraph.html}} React and ReactDOM are distributed as CommonJS modules by default. But what if you want to use them directly in the browser, with native ES modules and an importmap? In this note, I will show you how to create a clean, minified ESM bundle of React and ReactDOM using Rollup - and do it programmatically using Rollup's JavaScript API. {{/markup-paragraph.html}} {{>>markup-heading.html}}Step 1: Set Up Your Project{{/markup-heading.html}} {{>>markup-paragraph.html}} First, initialize your project: {{/markup-paragraph.html}} {{>>markup-code.html language="bash"}} npm init -y {{/markup-code.html}} {{>>markup-paragraph.html}} Install the required dependencies: {{/markup-paragraph.html}} {{>>markup-code.html language="bash"}} npm install react react-dom npm install --save-dev rollup @rollup/plugin-node-resolve @rollup/plugin-commonjs {{/markup-code.html}} {{>>markup-heading}}Step 2: Entry Files for React and ReactDOM{{/markup-heading.html}} {{>>markup-paragraph.html}} Since we want to treat React as a module, we will create two small entry files: {{/markup-paragraph.html}} {{>>markup-code.html language="javascript"}} // react.js import * as React from "react"; export * from "react"; export default React; {{/markup-code.html}} {{>>markup-code.html language="javascript"}} // react-dom.js import * as ReactDOM from "react-dom"; export * from "react-dom"; export default ReactDOM; {{/markup-code.html}} {{>>markup-paragraph.html}} These files serve as the input points for our Rollup bundles. {{/markup-paragraph.html}} {{>>markup-heading.html}}Step 3: Rollup JavaScript API Script{{/markup-heading.html}} {{>>markup-paragraph.html}} Create a file named bundle-react.js: {{/markup-paragraph.html}} {{>>markup-code.html language="javascript"}} import fs from "node:fs"; import path from "node:path"; import {rollup} from "rollup"; import resolve from "@rollup/plugin-node-resolve"; import commonjs from "@rollup/plugin-commonjs"; // bundles to generate const bundles = [ { input: "react.js", output: "dist/react.esm.js", }, { input: "react-dom.js", output: "dist/react-dom.esm.js", }, ]; // generate all bundles const allBundlesPromises = bundles.map(bundle => { const config = { input: bundle.input, plugins: [ resolve(), commonjs(), ], }; return rollup(config) .then(result => { return result.write({ file: bundle.output, format: "esm", }); }) .then(() => { console.log(`âœ… Built ${bundle.output}`); }); }); Promise.all(allBundlesPromises).catch(error => { console.error(error); }); {{/markup-code.html}} {{>>markup-heading.html}}Step 4: Run the Script{{/markup-heading.html}} {{>>markup-paragraph.html}} Add this to your package.json: {{/markup-paragraph.html}} {{>>markup-code.html language="json"}} "scripts": { "build:esm": "node bundle-react.js" } {{/markup-code.html}} {{>>markup-paragraph.html}} And then run: {{/markup-paragraph.html}} {{>>markup-code.html language="bash"}} npm run build:esm {{/markup-code.html}} {{>>markup-heading.html}}Step 5: Use with Import Maps in the Browser{{/markup-heading.html}} {{>>markup-paragraph.html}} Here is how you can use your ESM bundles in a browser-native environment: {{/markup-paragraph.html}} {{>>markup-code.html language="html"}}
